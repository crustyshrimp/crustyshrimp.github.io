Arduino

Exercise 13: TinkerCad




Exercise 13, wired up a LED to pin 8 and program a blink function onto it, learned how to calculate resistance using the V=IR formula, and to read resistor value based on the colour of the bands.

Exercise 14: 7 Segment LED Display




Exercise 14, wired up a 7 segment display to switches and resistors. The different switches connected to different pins of the display and when each switch controls one of the segments on the display, different combinations allows you to create different numbers.

Exercise 15 & 16: 7 Segment LED Display Countdown Timer& LED Counter using the Serial Monitor






Exercise 15 and 16 combined, as the LED blinks, the counter counts the number of blinks, and displays it on the serial monitor. Uses count function, different cases are configured to the corresponding number. (eg. in case 1, segments are configured to display number 1 on display).

Exercise 17: Matrix Keypad




Exercise 17, coded the matrix keypad. Pulling up serial monitor, will display a "starting..." text, after pressing a number, the number will be displayed on the serial monitor.
