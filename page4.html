Arduino

Exercise 13: TinkerCad




Exercise 13, wired up a LED to pin 8 and program a blink function onto it, learned how to calculate resistance using the V=IR formula, and to read resistor value based on the colour of the bands.

Exercise 14: 7 Segment LED Display




Exercise 14, wired up a 7 segment display to switches and resistors. The different switches connected to different pins of the display and when each switch controls one of the segments on the display, different combinations allows you to create different numbers.

Exercise 15 & 16: 7 Segment LED Display Countdown Timer& LED Counter using the Serial Monitor






Exercise 15 and 16 combined, as the LED blinks, the counter counts the number of blinks, and displays it on the serial monitor. Uses count function, different cases are configured to the corresponding number. (eg. in case 1, segments are configured to display number 1 on display).

Exercise 17: Matrix Keypad




Exercise 17, coded the matrix keypad. Pulling up serial monitor, will display a "starting..." text, after pressing a number, the number will be displayed on the serial monitor.

Exercise 18: LDR Photoresistor




Exercise 18, wired a LDR resistor and LED setup, LDR detects if the light value is above 512, if it is it will make the LED light up, if it is below 512, it will make the LED turn off.

Exercise 19: Ultrasonic Sensor




Exercise 19, ultrasonic sensor is wired to pin 9 and 10, trigpin and echopin respectively. Distance in cm is printed on the serial monitor.

Exercise 20: PIR Motion Sensor




Exercise 20, wired a PIR sensor, whenever a motion is detected it prints on the serial monitor. Whenever it detects a subsequent motion, the starts counting on the serial monitor.

Exercise 21: Temperature Sensor




Exercise 21, wired a temperature sensor, whenever the temperature changes, it starts to print the current temperature in both degree celsius and fahrenheit on the serial monitor.

Exercise 21: DHT Sensor




Exercise 21, wired a DHT sensor, whenever the temperature or humidity changes, it starts to print the current temperature in both degree celsius, fahrenheit and also the humidity in percentage on the serial monitor.

Exercise 22: Potentiometer




Exercise 22, wired a potentiometer and LED. Whenever the potentiometer value is more than 1023, the LED will light up. When the potentiometer value is less than 1023, the LED will turn off.
